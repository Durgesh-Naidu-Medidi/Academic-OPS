# Academic Ops

Academic Ops is a college management system developed using Flask and MySQL.

## Features

- Student Management
- Attendance Management
- Marks Entry
- Result Report
- Attendance Report
- Faculty Module

## Tech Stack

- Python
- Flask
- MySQL
- HTML
- CSS
- Bootstrap

## Project Purpose

Built as final year BTech CSE project to manage academic operations digitally.
